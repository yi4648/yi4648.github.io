什么是前端开发

前端开发是创建web页面或者app等前端界面呈现给用户的过程，通过HTML，CSS和JavaScript以及衍生出来的各种技术、框架、解决方案，来实现互联网产品的用户界面交互。

前端的三把斧

HTML5相当于HTML+CSS+JavaScript+API

HTML

文本标记语言（HyperText Markup Language）描述了网站的结构，是一种标记语言。一个HTML元素是一个HTML文件的一个基本组成单位。

CSS

层叠样式表（Cascading Style Sheets）主要职能就是确定布局和元素的表现。设定页面布设定页面元素样式、设定适用于所有网页的全局样式。使得页面的内容和样式得以分离。

JavaScript

一种高级的、解释型的编程语言，它能使网页可交互。添加动态功能（例如拥有复杂的动画、表单验证、页面特效、操作页面元素等）。支持面向对象编程、命令式编程、以及函数式编程。
